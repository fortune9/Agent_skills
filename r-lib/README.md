# r-lib Skills

Collection of skills for R package developers using the r-lib ecosystem.

## Available Skills

- **[learn-R](learn-R/)** - Learn R packages and functions using `btw::btw()` for AI agent context generation. This skill provides guidance on generating contextual prompts for learning R packages and functions with the btw package.

## Other Public Skills

For additional r-lib skills including testing, CLI development, CRAN checks, and more, see the [posit-dev/skills](https://github.com/posit-dev/skills) repository:

- **testing-r-packages** - Testing R packages with testthat 3rd edition
- **cli** - CLI styling and user communication with the cli package
- **cran-extrachecks** - CRAN submission preparation and extra checks
- **lifecycle** - R package lifecycle management
- **mirai** - Async and parallel computing with mirai
- **r-package-development** - General R package development workflows
